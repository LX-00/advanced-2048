How to play:

  • Press arrow / WASD / TFGH / IJKL keys or slide on your touch screen to move the tiles.
  • When two tiles with the same value touch, they merge into one and the value of the merged tile is the original value added one.
  • When a tile with the pattern "+=x" or "-=x" and another tile with value y touch, they merge into one and the value of the merged tile is x+y or x-y.
